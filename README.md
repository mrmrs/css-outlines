# css-outlines

Functional CSS for outlines

## Filesize

| File | Size |
|------|------|
| `dist/outlines.css` | 7544 bytes |
| `dist/outlines.min.css` | 5344 bytes (932 Gzipped) |

## Install

```sh
npm install css-outlines
```

## Usage

### Import

```css
@import "css-outlines";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-outlines/dist/outlines.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-outlines/dist/outlines.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.oo--blk-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/outlines.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/outlines.css` — formatted
- `dist/outlines.min.css` — minified
